about

The about command displays information of the PMON/2000 project, how it came to be ported to the PowerPC platform and who contributed. The amount of information displayed will vary across implementations, based on who contributes features to the project.

There are some useful URL's on 'about' screen that are resources for more information about PMON/200 and releated projects.

Format

The format for the reboot command is:
PMON> about
