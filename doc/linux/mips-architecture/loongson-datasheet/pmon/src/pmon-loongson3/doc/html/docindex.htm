PMON/2000 Commands by Name

PMON/2000 (running on the Target) provides a set of powerful assembly-level debugging commands.

Click here to see an example annotated debug session.
 
Command Function Execution
Control		 Display &
Modify		 Environment Misc Diagnostics
about About PMON/2000

boot Boot command (net or scsi)
bt Stack backtrace
c Continue
call Call
copy Copy
d Display
date Display/set date and time
db Delete breakpoint
debug Debug
dump Upload via RS232/Ethernet
eset Edit variable
fill Fill
flash Write or Erase a flash device area  
flush Flush data and/or instruction cache
g Go
h Help
hi History
l List (disassemble)
load Download via RS232/Ethernet
ls List symbols
m Memory display/modify
more More
mt Memory test
netboot Network boot load
ping Network setup test
r Register display/modify
reboot reboot PMON
search Search
scsiboot SCSI Disk boot load
set Set variable
sh Command shell
stty Display/set terminal options
sym Set symbolic name
t Trace (single step)
to Trace (step over)
tr Transparent mode
unset Remove variable
version Display PMON/2000 Version
