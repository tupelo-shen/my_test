Flash Memory Types

The flash memory must support sector erase if it is to be used to hold configuration settings.

PMON/2000 is currently aware of the following devices that support sector, block or whole device erase.

 
AMD Am29F010 - 128KB
Am29F040 - 512KB
Atmel AT29F040 - 512KB
INTEL TE28F016SA - OnBoard 8MB
TE28F160S3 - OnBoard 8MB
