PMON 2000 Targets

PMON 2000 currently supports SBS Communications and Embedded Computer Technology PowerPC boards as Targets.

Each PMON 2000 target needs to be equipped with a set of suitably configured PMON 2000 BootROMs..

This package currently builds for the following boards:

SBS Embedded Computer Power•4e
SBS Embedded Computer Power•4C
SBS Communications Palomar-II
SBS Communications K2
SBS Communications Denali
