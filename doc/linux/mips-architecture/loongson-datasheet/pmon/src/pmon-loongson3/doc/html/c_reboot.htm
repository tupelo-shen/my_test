reboot

The reboot command attempts to restart the PMON/2000 monitor (and any other code which runs before PMON/2000 at bootstrap time) by jumping to 0xfff00100 - the PowerPC restart location.

If your system initialization depends on some device receiving a hardware reset, this may not work.

Format

The format for the reboot command is:
PMON> reboot
